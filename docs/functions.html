simulations.functions.test_V(params=None, trials=100, no_random=10, n=200, l=12, k=3, add_radial_search=False, additional_best_no=0, search=5, less_ability_coef=None, no_less=0, alternative_stop=False, stop=None, latex=True, analysis=False, delta_rho=None, open_excel=True) → None[source]#

Test a fixed V taking several trials for the random group.

Parameters:

  • params (tuple) – Parameters for the deterministic \(V\) function. If entered, the random \(V\) functions are ignored, it returns the results with just one function \(V\).

  • trials (int) – Number of trials for the random group.

  • no_random (int) – Number of elements in the random group.

  • n (int) – Number of elements in the function \(V\).

  • l (int) – Maximum number that each element in the heuristic can take.

  • add_radial_search (bool) – Flag to add a radial search to the pool of best phis.

  • additional_best_no (int) – Number of additional best phi functions to add to the pool of best phis.

  • search (int) – Number of elements in the region for the additional phis.

  • less_ability_coef (float) – Less ability coefficient.

  • no_less (int) – Number of less ability elements.

  • alternative_stop (bool) – Flag to use the alternative stop condition (stop when a disagreement cycle is detected).

  • stop (int) – Stop condition.

  • latex (bool) – Flag to create a \(\LaTeX\) table with the results.

  • analysis (bool) – Flag to analyze the phi functions.

  • delta_rho (list) – List of deltas and rhos for the group dynamics.

Excel Functions#

Functions related to Excel files.

simulations.functions.excel_results(n, l, k, no_Vs=100, latex=True, percentage_random=1, no_random=10, add_radial_search=False, additional_best_no=0, params=None, search=5, less_ability_coef=None, no_less=0, alternative_stop=False, stop=None, delta_rho=None, analysis=False, function=None, analysis_ability_difference=False, open_excel=True) → None[source]#

Create an Excel file with the results of the expected values and the number of iterations for each phi function.

Parameters:

  • no_Vs (int) – Number of \(V\) functions to test.

  • latex (bool) – Flag to create a :LaTeX: table with the results.

  • percentage_random (float) – Percentage of random \(V\) functions.

  • add_radial_search (bool) – Flag to add a radial search to the pool of best phis.

  • additional_best_no (int) – Number of additional best phi functions to add to the pool of best phis.

  • params – Parameters for the deterministic V function. If entered, the random V functions are ignored, it returns the results with just one function V.

  • search (int) – Number of elements in the region for the additional phis.

  • less_ability_coef (float) – The less ability coefficient. See simulator_dynamics.PhiFunction.

  • no_less (int) – The number of less ability elements.

  • alternative_stop (bool) – Flag to use the alternative stop condition (stop when a disagreement cycle is detected).

  • stop (int) – The stop condition.

  • analysis_ability_difference (bool) – Flag to analyze the ability difference.

  • delta_rho (list) – List of deltas and rhos for the group dynamics. In particular, if provided, it contains: - delta_rho[0]: The ratio of deltas and rhos for the random group. - delta_rho[1]: The ratio of deltas and rhos for the best group.

  • analysis (bool) – Flag to analyze the phi functions.

  • function (function) – The function \(V\).

  • open_excel (bool) – Flag to open the Excel file.

Returns:

None
